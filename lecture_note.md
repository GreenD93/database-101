### Query
```SQL
SELECT `칼럼의 목록`
FROM `테이블의 목록`
WHERE `검색조건`
```

```SQL
DELETE FROM `테이블 이름` WHERE `조건`
```

```SQL
INSERT INTO `테이블명` (`컬럼 목록`) VALUES `대응되는 입력값`
```

```SQL
UPDATE `테이블명` SET `컬럼명` = `대응되는 입력값` WHERE `조건`
```
